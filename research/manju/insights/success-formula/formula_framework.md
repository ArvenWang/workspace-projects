## 二、人物设定公式

### 主角公式
```
讨喜主角 = 高共情点 + 成长空间 + 金手指/特殊能力
```

### 反派公式
```
有效反派 = 清晰动机 + 强大能力 + 与主角的强关联
```

### CP公式
```
爆款CP = 强烈化学反应 + 外在阻碍 + 内在成长
```

## 三、情绪曲线公式

### 标准情绪曲线
```
起（开篇吸引）→ 承（关系建立）→ 转（冲突爆发）→ 合（高潮解决）
     ↓              ↓              ↓              ↓
   好奇/悬疑      甜蜜/投入      虐心/愤怒      爽快/满足
```

### 爽点密度
- 每3-5集一个小爽点
- 每10-15集一个大反转
- 每30集一个阶段性高潮

## 四、平台适配公式

### 抖音爆款公式
```
抖音爆款 = 强视觉冲击 + 3秒钩子 + 快节奏剪辑 + 洗脑BGM
```

### 快手爆款公式
```
快手爆款 = 下沉市场共鸣 + 真实感 + 接地气人设 + 家长里短
```

### 微信付费公式
```
微信爆款 = 强付费卡点 + 悬念堆积 + 情绪拉扯 + 身份揭秘
```

## 五、制作成本公式

### 成本结构
- 剧本：10-15%
- 演员：30-40%
- 拍摄：30-35%
- 后期：15-20%

### ROI公式
```
ROI = (播放量 × CPM + 付费收入 + 广告收入) / 制作成本
```

爆款标准：ROI > 3

## 六、选题公式

### 爆款选题矩阵
| 维度 | 选项A（高） | 选项B（中） | 选项C（低） |
|------|------------|------------|------------|
| 情绪强度 | 强爽/强虐 | 轻度甜/虐 | 平淡 |
| 新鲜感 | 反套路 | 微创新 | 老套路 |
| 共鸣度 | 普世情感 | 特定人群 | 小众 |
| 制作难度 | 低成本 | 中等 | 高成本 |

爆款组合：至少3个维度选高

## 七、标题公式

### 高点击标题要素
1. 数字（如：99次、3天、100亿）
2. 身份标签（如：总裁、弃妇、神医）
3. 强烈对比（如：丑女变女神、穷小子变首富）
4. 情绪词汇（如：虐心、甜炸、爽翻）
5. 悬念词（如：没想到、竟然是、真相是）

示例公式：
```
[身份标签] + [极端处境] + [反转暗示]
例：被抛弃的丑女，3天后让总裁跪地求饶
```

## 八、开篇公式

### 标准开篇结构
```
第1秒：视觉冲击/悬念场景
第3秒：主角处境/核心问题
第10秒：冲突爆发/选择困境
第30秒：钩子抛出/付费引导
```

## 九、节奏公式

### 信息密度
- 每30秒必须有新信息
- 每集必须有情绪变化
- 每3集必须有情节推进

### 反转密度
- 小反转：每5-10集
- 中反转：每20-30集
- 大反转：第40集、第80集
