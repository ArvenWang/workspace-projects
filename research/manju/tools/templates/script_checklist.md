# 剧本分析检查清单

> 用于评估和优化漫剧剧本的实用工具
> 
> 使用方法：创作或评估时逐项检查，确保不遗漏关键要素

---

## 一、开篇检查（第1集）

### 黄金3秒 ✅
- [ ] 是否有强视觉冲击？（高颜值/激烈冲突/精美画面）
- [ ] 是否抛出悬念/疑问？
- [ ] 是否建立情绪钩子？（让人好奇/愤怒/期待）

### 10秒留人 ✅
- [ ] 主角处境是否清晰？（困境/特殊身份）
- [ ] 核心冲突是否建立？
- [ ] 观众是否产生投入感？

### 第1集完整性 ✅
- [ ] 主角背景是否交代清楚？
- [ ] 当前困境是否令人同情？
- [ ] 是否埋下后续伏笔？
- [ ] 结尾是否有小钩子？

---

## 二、人物设定检查

### 女主检查 ✅
- [ ] 是否有高共情点？（被欺负/善良/努力）
- [ ] 是否有成长空间？（从弱到强）
- [ ] 是否有特殊能力/金手指？（医术/武功/智谋）
- [ ] 是否值得被喜欢？
- [ ] 是否有记忆点？（不脸谱化）

### 男主检查 ✅
- [ ] 是否足够强大？（有钱/有权/有才华）
- [ ] 对女主是否专一/独宠？
- [ ] 是否有反差萌？（对外冷酷，对女主温柔）
- [ ] 是否有魅力点？（高颜值/好身材/独特气质）
- [ ] 与女主是否有化学反应？

### 反派检查 ✅
- [ ] 动机是否清晰？（为什么针对主角）
- [ ] 能力是否足够强？（给主角造成威胁）
- [ ] 与主角是否有强关联？（不是凭空出现）
- [ ] 是否令人讨厌？（观众想看他被打脸）
- [ ] 下场是否够惨？（解气）

### 配角检查 ✅
- [ ] 助攻型配角是否有功能？（帮助主角）
- [ ] 搞笑型配角是否适度？（不抢戏）
- [ ] 没有多余人物？（每个都有作用）

---

## 三、剧情结构检查

### 整体结构 ✅
- [ ] 是否遵循100集标准结构？
- [ ] 开篇（1-5集）是否完成引流任务？
- [ ] 发展阶段（6-50集）是否有足够内容？
- [ ] 高潮部分（51-80集）是否密集？
- [ ] 收尾（81-100集）是否圆满？

### 节奏检查 ✅
- [ ] 每集是否都有新信息/进展？
- [ ] 每3集是否都有小爽点/情绪点？
- [ ] 每10集是否都有身份/真相揭露？
- [ ] 没有水剧情？（每集都值得看）
- [ ] 没有逻辑硬伤？

### 付费卡点检查 ✅
- [ ] 第10集左右是否有第一个强付费点？
- [ ] 第30集是否有中高潮/大误会？
- [ ] 第50集是否有阶段性高潮？
- [ ] 第80集是否有终极危机？
- [ ] 每个付费点是否都有：悬念+高情绪+投入度？

---

## 四、情绪设计检查

### 情绪曲线 ✅
- [ ] 整体曲线是否合理？（不是一直平或一直虐）
- [ ] 虐点后是否有爽点？（不让观众憋屈太久）
- [ ] 爽点是否够爽？（打脸够狠/宠爱够甜）
- [ ] 高潮部分情绪是否达到顶点？

### 爽点检查（女频）✅
- [ ] 是否有被独宠/偏爱？
- [ ] 是否有身份揭露？（原来她是...）
- [ ] 是否有打脸渣男/绿茶？
- [ ] 是否有从被欺负到被尊重？
- [ ] 是否有真相大白？

### 爽点检查（男频）✅
- [ ] 是否有身份揭露？（原来他是...）
- [ ] 是否有用实力碾压对手？
- [ ] 是否有护短（保护家人/爱人）？
- [ ] 是否有金钱/权力展示？
- [ ] 打脸是否够狠？

### 虐点检查 ✅
- [ ] 误会是否合理？（不是强行误会）
- [ ] 虐是否为后续爽做铺垫？
- [ ] 虐是否过度？（避免观众弃剧）
- [ ] 虐后补偿是否够甜/够爽？

---

## 五、台词与细节检查

### 台词检查 ✅
- [ ] 是否有洗脑金句？（易传播）
- [ ] 台词是否符合人物身份？
- [ ] 是否有装X台词？（男频）
- [ ] 是否有甜宠台词？（女频）
- [ ] 是否有反派经典语录？

### 名场面检查 ✅
- [ ] 是否有至少3个记忆点名场面？
- [ ] 名场面是否适合截图传播？
- [ ] 名场面是否有BGM配合？

---

## 六、商业化检查

### 付费转化 ✅
- [ ] 付费点位置是否合理？
- [ ] 付费前铺垫是否足够？（观众已投入）
- [ ] 付费后的内容是否值得？（不让观众后悔）
- [ ] 是否有复购设计？（不止付一次）

### 广告植入 ✅（如需要）
- [ ] 植入是否自然？
- [ ] 是否影响观看体验？
- [ ] 是否符合剧情逻辑？

---

## 七、平台适配检查

### 抖音适配 ✅
- [ ] 前3秒是否够抓人？
- [ ] 是否有高颜值画面？
- [ ] 节奏是否够快？
- [ ] 是否适合竖屏观看？

### 快手适配 ✅
- [ ] 是否足够接地气？
- [ ] 爽感是否直接？
- [ ] 是否适合下沉市场？

### 微信适配 ✅
- [ ] 付费点是否够强？
- [ ] 悬念是否够密集？

---

## 八、创新度检查

### 反套路 ✅
- [ ] 是否有创新点？（不是完全复制）
- [ ] 老套路是否有新包装？
- [ ] 是否有独特记忆点？

### 避免雷区 ✅
- [ ] 没有抄袭争议？
- [ ] 没有三观问题？（如美化家暴）
- [ ] 没有明显BUG？

## 十、常见问题检查

### 逻辑问题
- [ ] 时间线是否清晰？（没有混乱）
- [ ] 人物行为是否符合逻辑？
- [ ] 没有明显的BUG？

### 节奏问题
- [ ] 没有水剧情？
- [ ] 没有过快过慢？
- [ ] 高潮部分足够密集？

### 人物问题
- [ ] 人物不脸谱化？
- [ ] 人物有成长/变化？
- [ ] 人物关系清晰？

### 商业问题
- [ ] 付费点够吸引人？
- [ ] 后续内容支撑得起付费？
- [ ] 有传播点？

---

## 使用建议

1. **创作阶段**：每写完10集，用此清单检查一次
2. **修改阶段**：针对低分项重点优化
3. **评估阶段**：多人独立打分，取平均值
4. **复盘阶段**：上线后根据数据反馈，反向验证清单有效性
