## 🎬 什么是"漫剧"？

**定义**：漫剧（又称"短剧"、"小程序剧"、"竖屏剧"）是指
- 单集时长：1-5分钟
- 总集数：80-100集
- 观看平台：抖音、快手、微信小程序、B站等
- 题材：霸总、甜宠、复仇、穿越、重生等强情绪类型
- 商业模式：付费解锁、广告植入、IP衍生

**市场规模**：
- 2024年短剧市场规模预计突破500亿
- 抖音短剧日活用户超2亿
- 微信小程序短剧付费用户超千万

## 📊 研究范围

### 时间范围
- **主要周期**：2024年1月 - 2025年12月（两年）
- **数据来源**：各平台榜单、行业报告、媒体报道

### 平台覆盖
| 平台 | 榜单来源 | 特点 |
|------|----------|------|
| 抖音 | 短剧热榜 | 流量最大 |
| 快手 | 星芒短剧 | 下沉市场 |
| 微信 | 小程序短剧榜 | 付费转化高 |
| B站 | 短剧分区 | 年轻用户 |
| 红果 | 免费短剧榜 | 番茄旗下 |
| 番茄小说 | 衍生短剧 | IP改编 |

### 分析维度

#### 1. 基础数据层
- [ ] 剧名、集数、时长
- [ ] 播放平台、上线时间
- [ ] 播放量、点赞数、评论数
- [ ] 付费转化率（如有数据）

#### 2. 内容分析层
- [ ] **剧情结构**：黄金3秒、卡点设计、高潮分布
- [ ] **人物设定**：主角人设、反派设定、配角功能
- [ ] **情绪曲线**：爽点分布、虐点设计、反转节奏
- [ ] **题材类型**：霸总、甜宠、复仇、穿越等

#### 3. 创作要素层
- [ ] **标题技巧**：如何取吸睛标题
- [ ] **开篇设计**：前3秒如何留住观众
- [ ] **钩子设置**：每集结尾如何引导付费
- [ ] **金句提炼**：爆款台词、洗脑slogan

#### 4. 商业要素层
- [ ] 制作成本 vs 收益
- [ ] 演员选择（素人 vs 明星）
- [ ] IP衍生（小说改编 vs 原创）
- [ ] 变现模式（付费解锁 vs 广告植入）

## 📊 数据分析框架

### 爆款指数模型

```
爆款指数 = 播放量权重(30%) 
         + 互动率权重(20%) 
         + 付费转化权重(30%) 
         + 长尾效应权重(20%)
```

### 内容质量评估维度

| 维度 | 权重 | 评估指标 |
|------|------|----------|
| 开篇吸引力 | 20% | 3秒留存率、前10秒完播率 |
| 情节紧凑度 | 25% | 卡点密度、无尿点时长 |
| 情绪感染力 | 25% | 评论情感分析、分享率 |
| 人物魅力 | 15% | 角色讨论度、同人创作 |
| 制作质量 | 15% | 画面、演技、服化道 |
