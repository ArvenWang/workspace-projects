# 六视角决策系统

> 应用X学到的多智能体研究系统

## 6个视角

1. **用户视角** - 用户真正需要什么？
2. **技术视角** - 能不能做？难点在哪？
3. **商业视角** - 有什么价值？
4. **风险视角** - 有什么风险？
5. **执行视角** - 怎么落地？
6. **长期视角** - 未来会怎样？

## 决策流程

### Phase 1: 独立分析
每个视角独立思考

### Phase 2: 交叉质询
- 用户说要做 vs 技术说做不到 → 冲突
- 商业说能赚 vs 风险太大 → 冲突

### Phase 3: 裁决
综合所有视角，输出决策

## 应用场景

- 做不做这个案例？
- 先做哪个？
- 这个方案好不好？
